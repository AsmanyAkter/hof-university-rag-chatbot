# Data Collection Pipeline — Setup & Instructions

## Requirements

```bash
pip install -r requirements.txt
```

## Step-by-Step Instructions

> **Note:** Run notebooks in the following order, as each step depends on the output of the previous one.

1. `spo's_extraction.ipynb`
2. `data-scrape-full-programs-all-dept.ipynb`
3. `aspo_extraction.ipynb`
4. `general_info_extractions.ipynb`
5. `professor_info_scrape.ipynb`
6. `question-answer-generation.ipynb`

   

> **Note:** There are two data folders:
> - `data/` — contains the final processed data used in the RAG pipeline
> - `debug_data/` — contains intermediate data before final processing



### Final Output Files in "data" folder:

| File | Description |
|------|-------------|
| `all_programs_data.json` | 37 programs with full details and SPO data |
| `hof_university_staff.json` | 175 staff entries |
| `aspo_priority_sections.json` | 24 ASPO sections (high and medium priority) |
| `general_hof_university_data.json` | General university info including academic calendar, financing, admission requirements, and FAQs |

> `test_cases_100.csv` is generated by `question-answer-generation.ipynb` and contains 100 Q&A pairs used for the without-memory pipeline evaluation.
